

## 환경준비

- npm version = 6.13.4
- Node v12.14.1
- go version >=1.17.x
## 설치 

```zsh
git clone https://github.com/jzu/wallet-site
npm install
```

## 환경설정파일


- ``VUE_APP_AVA_IP`` node ip 주소
- ``VUE_APP_AVA_PORT`` node port 
- ``VUE_APP_AVA_PROTOCOL`` http or https
- ``VUE_APP_NETWORK_ID`` 연결할 network id
- ``VUE_APP_CHAIN_ID``  연결할 블록체인 id

## 실행

```zsh
npm run serve
```
